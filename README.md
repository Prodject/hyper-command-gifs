![Demo of Plugin using git push command](example/git-push-demo.gif)

## How to use

Install [Hyper](https://hyper.is) and add `hyper-command-gifs`
to `plugins` in `~/.hyper.js`.

```javascript
plugins: [
  'hyper-command-gifs'
]
```
